# Shourya Auto World

## Features
- User-friendly interface for easy navigation.
- Comprehensive listing of automotive services.
- Detailed product descriptions and images.
- Search functionality to quickly find services and products.
- Admin panel for managing listings and user inquiries.
- Secure payment gateway integration.

## How to Use Admin Panel
1. Login with your admin credentials.
2. Navigate to the dashboard where you can see management options.
3. Use the sidebar to manage services and products.
4. Ensure to save changes after each update.

## Deployment Instructions
1. Clone the repository using `git clone https://github.com/gauravnagare32-dot/Shouryaautoworld.git`.
2. Navigate to the project directory.
3. Install dependencies using `npm install`.
4. To start the application, run `npm start`.
5. For production, build the project using `npm run build` and deploy the contents of the `build` folder.
